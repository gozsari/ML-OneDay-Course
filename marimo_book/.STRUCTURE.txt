🎯 QUICK START COMMANDS
-----------------------

1. Install dependencies:
   $ pip install -r requirements.txt

2. Launch the book:
   $ ./start_book.sh              [Linux/Mac]
   $ start_book.bat               [Windows]
   $ marimo edit 0-Index.py       [Direct]

3. Open a specific chapter:
   $ marimo edit 1-Introduction_to_Machine_Learning.py

4. Export to HTML:
   $ marimo export html 0-Index.py -o index.html
