# How to Create Environment

## Create a virtual environment:  
```sh
python3 -m venv .venv
source .venv/bin/activate
```
## Install dependencies:  
```sh
pip install -r requirements.txt
```
## Activate the environment:  
```sh
source .venv/bin/activate
```

## Deactivate the environment:  
```sh
deactivate
```
